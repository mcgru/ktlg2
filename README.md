# Описание
Утилита для раскладывания фоточек и видосиков по структуре папок, которая строится на базе имеющейся структуры с добавлением префикса года-месяца к именам папок.

Попробуйте на копии ваших файлов.
Сначала сделайте структуру файлов плоской:
`ktlg2 plane PATH`
а затем проведите организацию файлов:
`ktlg2 org[anize] PATH`
или просто
`ktlg2 PATH`

## Дополнительно

В конце работы утилита проставляет метку времени mtime для папок, равную дате/времени изменения последнего файла/папки внутри.

**Повторный запуск на ту же папку - не меняет финальную структуру папок и файлов!**
